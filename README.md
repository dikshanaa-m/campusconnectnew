 🌐 Campus Connect

Campus Connect is a web-based platform that helps students discover, join, and participate in college clubs. It also allows club administrators to manage members, events, and activities in an organized way.

 🚀 Project Description

Campus Connect solves the problem of scattered communication between students and clubs. It provides a single platform where all club-related activities can be managed efficiently.

Students can explore clubs and request to join, while admins can approve members and manage club operations.

 👥 User Roles
🎓 Student

* View all available clubs
* Send join requests
* Access club events and updates

 🧑‍💼 Club Admin

* Approve or reject join requests
* Manage club members
* Create and manage events, tasks, and resources

 ⚙️ Features

* 🔐 Secure login with role-based access
* 📩 Join request system
* 👥 Club member management
* 📅 Event management
* 📂 Tasks and resources sharing
* 🔄 Real-time database updates

 🛠️ Technologies Used

* Frontend: React.js
* Backend: Node.js & Express.js
* Database: MongoDB Atlas
* Authentication: JWT (JSON Web Token)


 🌍 Deployment

* Backend deployed on **Render**
* Frontend deployed on **Vercel**

 ⚡ Working Process

1. User logs into the system
2. Student browses clubs and sends join request
3. Admin receives and approves/rejects request
4. Approved students become club members
5. Admin manages events, tasks, and resources


🎯 Key Highlights

* Role-based authentication system
* Secure backend with JWT
* Separate collection for club members
* Real-time updates using MongoDB
* Clean and scalable architecture



🔮 Future Improvements

🙌 Conclusion

Campus Connect simplifies club management by providing a centralized system for both students and administrators, improving communication and engagement within the campus.
